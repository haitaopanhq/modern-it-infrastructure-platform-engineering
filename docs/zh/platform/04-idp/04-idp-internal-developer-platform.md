# 07-04: 一图看懂 IDP 内部开发平台

## 核心观点
IDP 是平台工程的载体，Backstage 是事实标准。

## Backstage 架构

```
┌─────────────────────────────────────────────────────────────┐
│                  Backstage 架构                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │              Backstage App                          │  │
│   │   ┌──────────┐ ┌──────────┐ ┌──────────────────┐   │  │
│   │   │ Catalog  │ │  TechDocs│ │  Plugins         │   │  │
│   │   │ (服务目录)│ │ (技术文档)│ │ (Scaffolder等)  │   │  │
│   │   └──────────┘ └──────────┘ └──────────────────┘   │  │
│   └─────────────────────────────┬───────────────────────┘  │
│                                 │                           │
│   ┌─────────────────────────────┼───────────────────────┐  │
│   │              Plugins                               │   │
│   │   github-actions │ jenkins │ ArgoCD │ PagerDuty    │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   核心功能:                                                 │
│   • 软件目录 (Software Catalog)                            │
│   • 技术文档 (TechDocs)                                    │
│   • 应用脚手架 (Scaffolder)                                │
│   • 可扩展插件生态                                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
