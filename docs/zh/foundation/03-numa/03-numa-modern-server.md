# 01-03: 一图看懂 NUMA 与现代服务器

## 核心观点
NUMA 架构是现代服务器的基础，不理解 NUMA 就无法优化性能。

## NUMA 架构图


## NUMA 感知实践

```bash
# 查看 NUMA 布局
numactl --hardware

# 查看进程 NUMA 位置
numastat -p <pid>

# 绑定到特定 Node
numactl --cpunodebind=0 --membind=0 ./process
```

## 云环境中的 NUMA

| 云产品 | NUMA 策略 | 注意事项 |
|--------|----------|----------|
| ECS | 默认绑定 | 需确认实例规格 |
| GCE | 自动感知 | 选择合适机器类型 |
| Azure | 可配置 | 启用 SR-IOV |

## 性能影响

- MySQL/PostgreSQL: 本地访问 vs 跨 Node 访问
- Kafka: 顺序读写与 NUMA 亲和
- Redis: 内存分配策略
