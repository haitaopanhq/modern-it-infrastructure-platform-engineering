# 01-04: 一图看懂 CPU → GPU → DPU 演进

## 核心观点
数据中心处理器演进：CPU → GPU → DPU，专用芯片时代到来。

## 演进图

```
┌─────────────────────────────────────────────────────────────┐
│                   处理器演进                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   时代1: CPU 时代                                           │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                    CPU                              │  │
│   │   通用计算 │ 控制 │ 缓存 │ 内存控制                  │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   时代2: CPU + GPU 时代                                     │
│   ┌──────────────┐      ┌─────────────────────────────┐   │
│   │     CPU      │ ←──→ │           GPU               │   │
│   │  通用计算    │      │   并行计算 (CUDA cores)     │   │
│   └──────────────┘      └─────────────────────────────┘   │
│                                                             │
│   时代3: CPU + GPU + DPU 时代                               │
│   ┌──────────┐  ┌──────────┐  ┌────────────────────────┐  │
│   │   CPU    │  │   GPU    │  │         DPU            │  │
│   │ 通用计算 │  │ AI计算   │  │ 数据处理/加速          │  │
│   └──────────┘  └──────────┘  │ • 网络卸载              │  │
│                               │ • 存储卸载              │  │
│                               │ • 安全卸载              │  │
│                               └────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## DPU 功能

| 功能 | 传统方式 | DPU 方式 |
|------|---------|----------|
| 网络 | CPU 处理 | DPU 卸载 |
| 存储 | CPU 处理 | DPU 卸载 |
| 安全 | 软件实现 | 硬件加速 |
| 虚拟化 | CPU 虚拟化 | DPU 卸载 |

## 代表产品

- **NVIDIA BlueField**: DPU 标杆
- **Intel IPU**: 基础设施处理器
- **AMD Pensando**: DPU 产品线
