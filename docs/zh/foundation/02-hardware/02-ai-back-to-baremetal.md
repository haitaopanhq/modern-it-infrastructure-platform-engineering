# 01-02: 一图看懂 AI 为什么重新回到裸金属

## 核心观点
AI 工作负载重新暴露了硬件瓶颈，裸金属成为 AI 基础设施首选。

## 对比图

```
┌─────────────────────────────────────────────────────────────┐
│               Web 工作负载 vs AI 工作负载                   │

│   ┌─────────────────┐                  ┌─────────────────┐ │
```

## 硬件指标对比

| 指标 | 传统 Web | AI 训练 | AI 推理 |
|------|----------|---------|---------|
| 延迟要求 | ms 级 | 无 | <100ms |
| 内存 | GB 级 | TB 级 | GB 级 |
| 带宽 | Gbps | Tbps | Tbps |
| 瓶颈 | 网络 | GPU/显存 | 显存 |

## 为什么回归裸金属

1. **虚拟化开销**：GPU 直通/ SR-IOV 需要裸金属
2. **显存访问**：PCIe 带宽不够，需要 NVLink
3. **延迟敏感**：分布式训练通信延迟影响收敛
4. **成本**：虚拟化 licensing 成本高
