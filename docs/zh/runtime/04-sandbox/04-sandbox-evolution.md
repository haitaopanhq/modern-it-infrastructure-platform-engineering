# 04-04: 一图看懂沙箱隔离演进

## 核心观点
沙箱技术从浏览器到容器，不断进化。

## 隔离技术演进

```
┌─────────────────────────────────────────────────────────────┐
│                    沙箱隔离演进                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   浏览器沙箱:                                               │
│   ┌─────────────┐  ┌─────────────┐  ┌─────────────┐        │
│   │   iframe    │  │   Chrome    │  │   WebAssembly│       │
│   │   隔离      │  │   进程隔离  │  │   沙箱      │        │
│   └─────────────┘  └─────────────┘  └─────────────┘        │
│                                                             │
│   容器沙箱:                                                 │
│   ┌─────────────┐  ┌─────────────┐  ┌─────────────┐        │
│   │  namespace  │  │   cgroups   │  │    seccomp  │        │
│   │   命名空间   │  │   资源限制  │  │   系统调用  │        │
│   └─────────────┘  └─────────────┘  └─────────────┘        │
│                                                             │
│   VM 沙箱:                                                  │
│   ┌─────────────┐  ┌─────────────┐  ┌─────────────┐        │
│   │ Firecracker │  │   vmm       │  │  cloud-hv   │        │
│   │  微虚拟机   │  │  虚拟机监视器│  │  Rust 实现  │        │
│   └─────────────┘  └─────────────┘  └─────────────┘        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 隔离层级

| 层级 | 技术 | 防御目标 |
|------|------|---------|
| 进程 | namespace | 资源隔离 |
| 系统调用 | seccomp | 权限控制 |
| 内核 | gVisor | 内核漏洞 |
| 硬件 | Kata | 完全隔离 |
