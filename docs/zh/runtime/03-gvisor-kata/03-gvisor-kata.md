# 04-03: 一图看懂 gVisor/Kata

## 核心观点
安全容器是介于容器和虚拟机之间的折中方案。

## 安全容器对比

```
┌─────────────────────────────────────────────────────────────┐
│                 安全容器对比                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   Docker 容器:           gVisor:           Kata:            │
│   ┌───────────┐        ┌───────────┐     ┌───────────┐     │
│   │ 共享内核   │        │ Sentry    │     │  轻量VM   │     │
│   │ 性能最好   │        │ (用户空间)│     │  独立内核 │     │
│   │ 隔离最弱   │        │ 拦截系统  │     │  强隔离   │     │
│   └───────────┘        │ 调用       │     └───────────┘     │
│                        └───────────┘                       │
│                                                             │
│   性能: 高 ←─────────────→ 低                               │
│   隔离: 低 ←─────────────→ 高                               │
│                                                             │
│   场景:                           场景:                     │
│   • 微服务                       • 多租户                   │
│   • 开发测试                     • 不可信工作负载           │
│   • 信任区                       • 安全隔离要求高           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
