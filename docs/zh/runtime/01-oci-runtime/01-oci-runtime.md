# 04-01: 一图看懂 OCI Runtime

## 核心观点
OCI 定义了容器标准，让容器生态可互换。

## OCI 规范

```
┌─────────────────────────────────────────────────────────────┐
│                    OCI 规范栈                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌───────────────────────────────────────────────────────┐ │
│   │              OCI Image Specification                   │ │
│   │   • 镜像格式 (Manifest, Layer, Config)                │ │
│   │   • 分布协议 (Distribution)                           │ │
│   └───────────────────────────────────────────────────────┘ │
│                            │                                │
│   ┌───────────────────────────────────────────────────────┐ │
│   │              OCI Runtime Specification                 │ │
│   │   • 容器生命周期                                       │ │
│   │   • 容器配置 (Linux namespaces, cgroups)              │ │
│   │   • 执行环境                                           │ │
│   └───────────────────────────────────────────────────────┘ │
│                            │                                │
│   ┌───────────────────────────────────────────────────────┐ │
│   │              OCI Distribution Specification            │ │
│   │   • 镜像拉取协议                                       │ │
│   │   • 认证机制                                           │ │
│   └───────────────────────────────────────────────────────┘ │
│                                                             │
│   实现: containerd, CRI-O, Docker, Podman                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 容器生命周期

```mermaid
create → start → stop → delete
```
