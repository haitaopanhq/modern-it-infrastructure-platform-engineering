# 04-02: 一图看懂 containerd 与 Docker

## 核心观点
containerd 是 Docker 的核心，Docker 是一套完整工具链。

## 关系图

```
┌─────────────────────────────────────────────────────────────┐
│                 Docker vs containerd                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   Docker:                                                   │
│   ┌─────────────────────────────────────────────────────┐  │
│   │  Docker CLI  │  Docker Daemon (dockerd)             │  │
│   │              │  ┌─────────────┐ ┌────────────────┐ │  │
│   │              │  │ containerd  │ │   BuildKit     │ │  │
│   │              │  │   (Runtime) │ │   (Build)      │ │  │
│   │              │  └─────────────┘ └────────────────┘ │  │
│   │              │           │                          │  │
│   │              │    ┌──────┴──────┐                  │  │
│   │              │    │ runc        │                  │  │
│   │              │    │ (容器运行)  │                  │  │
│   │              │    └─────────────┘                  │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   containerd:                                              │
│   ┌─────────────────────────────────────────────────────┐  │
│   │   containerd (独立项目)                              │  │
│   │   • 镜像管理 (CRI)                                   │  │
│   │   • 容器生命周期                                     │  │
│   │   • 存储管理                                         │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 功能对比

| 功能 | Docker | containerd |
|------|--------|------------|
| 镜像构建 | ✅ | ❌ |
| 镜像拉取 | ✅ | ✅ |
| 容器运行 | ✅ | ✅ |
| 网络管理 | ✅ | ❌ |
| CLI 工具 | docker | ctr, nerdctl |
