# 03-02: 一图看懂 Kubernetes 为什么统一云原生

## 核心观点
Kubernetes 成功的关键是生态，而非技术本身。

## 成功因素

1. **Google Borg 遗产**: 大量生产验证
2. **声明式 API**: 基础设施即代码
3. **CNCF 治理**: 厂商中立
4. **生态完整**: 覆盖所有场景
5. **云厂商支持**: 全部拥抱
