# 03-03: 一图看懂公有云控制面

## 核心观点
云厂商真正卖的是控制面，而非资源。

## 控制面架构

```
┌─────────────────────────────────────────────────────────────┐
│                    公有云控制面                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   用户视角:                                                 │
│   ┌─────────────────────────────────────────────────────┐  │
│   │   API/Console → 控制面 → 数据面 (虚拟机/容器)       │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   云厂商视角:                                               │
│   ┌─────────────────────────────────────────────────────┐  │
│   │   控制面 (Cloud):                                    │  │
│   │   • 认证 (IAM)                                       │  │
│   │   • 计量 (Billing)                                   │  │
│   │   • 编排 (Orchestration)                            │  │
│   │   • 治理 (Governance)                               │  │
│   └─────────────────────────────────────────────────────┘  │
│                           │                                 │
│                           ▼                                 │
│   ┌─────────────────────────────────────────────────────┐  │
│   │   数据面 (Tenant):                                   │  │
│   │   • 虚拟机/容器                                     │  │
│   │   • 存储卷                                          │  │
│   │   • 网络                                            │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   用户为控制面付费: 按 API 调用次数/管理资源数收费          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 控制面服务

| 服务 | 功能 |
|------|------|
| IAM | 身份认证 |
| CloudTrail | 审计日志 |
| Config | 合规检查 |
| CloudFormation | 基础设施编排 |
| Resource Groups | 资源管理 |
