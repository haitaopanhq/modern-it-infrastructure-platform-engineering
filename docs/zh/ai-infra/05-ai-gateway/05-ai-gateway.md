# 09-05: 一图看懂 AI Gateway

## 核心观点
AI Gateway 是 AI 时代的 API Gateway。

## AI Gateway 功能

```
┌─────────────────────────────────────────────────────────────┐
│                   AI Gateway                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                   AI Gateway                         │  │
│   │  ┌───────────┐ ┌───────────┐ ┌───────────┐         │  │
│   │  │  模型路由  │ │  负载均衡  │ │   限流    │         │  │
│   │  │(多模型切换)│ │ (流量分发) │ │ (Token控制)│         │  │
│   │  └───────────┘ └───────────┘ └───────────┘         │  │
│   │  ┌───────────┐ ┌───────────┐ ┌───────────┐         │  │
│   │  │  缓存     │ │   监控    │ │   安全    │         │  │
│   │  │(Semantic) │ │ (LLM指标) │ │(内容审核) │         │  │
│   │  └───────────┘ └───────────┘ └───────────┘         │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   核心功能:                                                  │
│   • 请求路由 (基于模型/成本/负载)                           │
│   • Token 限流 (rpm/tpm)                                    │
│   • 响应缓存 (减少重复调用)                                 │
│   • 成本分摊 (按 Token 计费)                                │
│   • A/B Testing (模型对比)                                  │
│                                                             │
│   产品: Cloudflare AI Gateway, Portkey, Baseten            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
