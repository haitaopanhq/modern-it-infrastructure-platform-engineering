# 09-03: 一图看懂 Ray

## 核心观点
Ray 是分布式 AI 的基础设施。

## Ray 架构

```
┌─────────────────────────────────────────────────────────────┐
│                     Ray 架构                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │              应用层 (Ray Libraries)                 │  │
│   │   RLlib │ Tune │ SCRIPT │ Dataset │ Serve          │  │
│   └─────────────────────────────┬───────────────────────┘  │
│                                 │                           │
│   ┌─────────────────────────────┼───────────────────────┐  │
│   │              Core (Ray Core)                        │  │
│   │   ┌─────────┐ ┌─────────┐ ┌─────────────────────┐  │  │
│   │   │  GCS    │ │  Plasma │ │    Task Scheduler   │  │  │
│   │   │(全局控制)│ │ (对象存储)│ │   (分布式调度)     │  │  │
│   │   └─────────┘ └─────────┘ └─────────────────────┘  │  │
│   └─────────────────────────────┬───────────────────────┘  │
│                                 │                           │
│   ┌─────────────────────────────┴───────────────────────┐  │
│   │              集群层 (Ray Cluster)                    │  │
│   │   ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐      │  │
│   │   │Head Node│ │Worker 1│ │Worker 2│ │Worker N│      │  │
│   │   └────────┘ └────────┘ └────────┘ └────────┘      │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   核心概念:                                                  │
│   • Task: 远程函数                                          │
│   • Actor: 有状态服务                                       │
│   • Object: 分布式对象                                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
