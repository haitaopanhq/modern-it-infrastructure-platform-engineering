# 08-01: 一图看懂监控演进史

## 核心观点
监控从基础设施到应用再到业务，边界不断扩展。

## 演进图

```
┌─────────────────────────────────────────────────────────────┐
│                    监控演进史                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   第一代: 基础设施监控                                       │
│   Nagios → Zabbix → 主机/网络/存储                         │
│                                                             │
│   第二代: 应用性能监控 (APM)                                │
│   Dynatrace → New Relic → 追踪/指标/日志                   │
│                                                             │
│   第三代: 云原生监控                                         │
│   Prometheus → Grafana → 多维度指标                        │
│                                                             │
│   第四代: 可观测性 (Observability)                          │
│   OTel → 全链路追踪 → 统一可观测平台                        │
│                                                             │
│   关注点变化:                                               │
│   基础设施 → 应用 → 业务 → 体验                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
