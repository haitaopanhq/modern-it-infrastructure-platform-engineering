# 08-03: 一图看懂 eBPF 可观测性

## 核心观点
eBPF 实现了无侵入的可观测性。

## eBPF 可观测性工具

```
┌─────────────────────────────────────────────────────────────┐
│                 eBPF 可观测性工具                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                    eBPF                              │  │
│   │  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐   │  │
│   │  │   网络追踪  │ │   系统追踪   │ │  应用追踪   │   │  │
│   │  │   (Cilium)  │ │  (BCC/bpftrace)│ │ (Pixie)   │   │  │
│   │  └─────────────┘ └─────────────┘ └─────────────┘   │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   优势:                                                     │
│   • 零侵入 (无需修改应用代码)                               │
│   • 细粒度 (函数级)                                        │
│   • 低开销 (内核执行)                                       │
│   • 动态 attach (运行时加载)                               │
│                                                             │
│   工具: Cilium, Falco, Pixie, bpftrace, BCC                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
