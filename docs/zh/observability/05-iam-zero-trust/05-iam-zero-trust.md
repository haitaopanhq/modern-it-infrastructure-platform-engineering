# 08-05: 一图看懂 IAM 与零信任

## 核心观点
零信任是安全范式的根本转变。

## 零信任架构

```
┌─────────────────────────────────────────────────────────────┐
│                    零信任架构                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   传统边界模型:                    零信任模型:              │
│   ┌─────────────────┐            ┌─────────────────┐       │
│   │   外部防火墙   │            │   永不信任     │       │
│   │   ┌───────────┐│            │   始终验证     │       │
│   │   │  内部网络  ││            │   ┌───────────┐│       │
│   │   │   (信任)   ││            │   │  每次请求  ││       │
│   │   └───────────┘│            │   │   需认证   ││       │
│   └─────────────────┘            │   └───────────┘│       │
│                                  └─────────────────┘       │
│   核心原则:                                                 │
│   • 永不信任，始终验证                                      │
│   • 最小权限访问                                            │
│   • 微分段隔离                                              │
│   • 持续监控                                                │
│                                                             │
│   技术: SPIFFE, OPA, Service Mesh mTLS                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
