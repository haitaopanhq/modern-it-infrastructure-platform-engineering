# 02-01: 一图看懂计算虚拟化演进

## 核心观点
计算虚拟化从硬件辅助到软件定义，不断演进。

## 虚拟化类型对比

| 类型 | 实现方式 | 性能 | 隔离性 |
|------|---------|------|--------|
| 全虚拟化 | 二进制翻译 | 中 | 高 |
| 半虚拟化 | 修改 Guest OS | 高 | 中 |
| 硬件辅助 | VT-x/AMD-V | 高 | 高 |
| 容器 | 共享内核 | 极高 | 低 |
