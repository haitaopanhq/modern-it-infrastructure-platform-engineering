# 02-04: 一图看懂资源池化历史

## 核心观点
资源池化是云计算的本质，从分散到统一。

## 池化演进

```
┌─────────────────────────────────────────────────────────────┐
│                    资源池化演进                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   阶段1: 单一资源池                                         │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                   计算资源池                         │  │
│   │   虚拟机1 │ 虚拟机2 │ 虚拟机3 │ 虚拟机4            │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   阶段2: 分离资源池                                         │
│   ┌─────────────┐ ┌─────────────┐ ┌─────────────┐         │
│   │ 计算资源池   │ │ 存储资源池   │ │ 网络资源池   │         │
│   └─────────────┘ └─────────────┘ └─────────────┘         │
│                                                             │
│   阶段3: 超融合 (HCI)                                       │
│   ┌─────────────────────────────────────────────────────┐  │
│   │     超融合集群 (计算+存储+网络统一管理)              │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   阶段4: 分布式云                                           │
│   ┌─────────────────────────────────────────────────────┐  │
│   │     跨地域资源池 (统一调度，多区域部署)              │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 池化技术

| 技术 | 池化对象 | 典型产品 |
|------|---------|---------|
| vSphere DRS | 计算 | VMware |
| Ceph | 存储 | RBD/CephFS |
| vSwtich | 网络 | Linux/vSphere |
| HCI | 全部 | Nutanix/vSAN |
