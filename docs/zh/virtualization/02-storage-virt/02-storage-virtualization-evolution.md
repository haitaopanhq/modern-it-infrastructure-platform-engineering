# 02-02: 一图看懂存储虚拟化演进

## 核心观点
存储虚拟化从 RAID 到软件定义存储，不断抽象。

## 演进图

```
┌─────────────────────────────────────────────────────────────┐
│                 存储虚拟化演进                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   本地存储 → 网络存储 → 软件定义存储 → 云存储               │
│                                                             │
│   ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐│
│   │  RAID   │    │   SAN   │    │   SDS   │    │   S3    ││
│   │  磁盘   │    │  光纤   │    │  Ceph   │    │ 对象    ││
│   │  阵列   │    │  通道   │    │  MinIO  │    │  存储   ││
│   └─────────┘    └─────────┘    └─────────┘    └─────────┘│
│                                                             │
│   特点:                     特点:                          │
│   • 硬件依赖               • 网络集中管理                  │
│   • 扩展性有限             • 协议: iSCSI/FC               │
│                            • 特点:                         │
│                            • 软件抽象                      │
│                            • 协议: S3/CSI                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 存储类型

| 类型 | 协议 | 场景 |
|------|------|------|
| 块存储 | iSCSI, FC, NVMe-oF | 数据库 |
| 文件存储 | NFS, SMB | 共享文件 |
| 对象存储 | S3, Swift | 非结构化数据 |
| 文件系统 | POSIX, HDFS | 大数据 |
