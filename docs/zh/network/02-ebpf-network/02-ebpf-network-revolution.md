# 05-02: 一图看懂 eBPF 网络革命

## 核心观点
eBPF 让内核变得可编程，网络和安全领域最先受益。

## eBPF 应用

```
┌─────────────────────────────────────────────────────────────┐
│                    eBPF 网络应用                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌──────────────────────────────────────────────────────┐ │
│   │                   eBPF                               │ │
│   │  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐   │ │
│   │  │  网络追踪   │ │  负载均衡   │ │  安全观测   │   │ │
│   │  │  (Cilium)   │ │  ( kube-proxy)│ │ (Falco)    │   │ │
│   │  └─────────────┘ └─────────────┘ └─────────────┘   │ │
│   └──────────────────────────────────────────────────────┘ │
│                                                             │
│   优势:                                                     │
│   • 无需修改内核源码                                        │
│   • 运行时加载                                              │
│   • 性能接近内核模块                                        │
│   • 安全可控 (验证器)                                       │
│                                                             │
│   产品: Cilium, Falco, Pixie, Katran                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 性能对比

| 方案 | 包处理 | CPU 使用 |
|------|--------|---------|
| iptables | 200K pps | 高 |
| eBPF | 10M+ pps | 低 |
