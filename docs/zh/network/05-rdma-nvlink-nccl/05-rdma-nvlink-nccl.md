# 05-05: 一图看懂 RDMA/NVLink/NCCL

## 核心观点
AI 集群的网络是计算瓶颈，RDMA 是答案。

## GPU 通信技术

```
┌─────────────────────────────────────────────────────────────┐
│                 GPU 集群通信技术                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   传统 TCP:                                                │
│   CPU ──► 网卡 ──► 内核 ──► 用户 ──► GPU                   │
│   │         │        │          │                         │
│   └─────────┴────────┴──────────┴────── 拷贝 4-5 次        │
│                                                             │
│   RDMA:                                                    │
│   GPU ──► RDMA ──► RDMA ──► GPU                            │
│   │                        │                               │
│   └────────────────────────┘ 零拷贝, 直接访问              │
│                                                             │
│   技术对比:                                                 │
│   ┌───────────┬──────────┬──────────┬────────────┐        │
│   │  技术     │  带宽    │  延迟    │   场景     │        │
│   ├───────────┼──────────┼──────────┼────────────┤        │
│   │  TCP/IP   │  100Gbps │  ~10us   │  传统网络  │        │
│   │  RoCE     │  200Gbps │  ~2us    │  GPU 集群  │        │
│   │  iWARP    │  100Gbps │  ~3us    │  跨机房    │        │
│   │  NVLink   │  900GB/s │  ~1us    │  单机多GPU │        │
│   └───────────┴──────────┴──────────┴────────────┘        │
│                                                             │
│   NCCL: NVIDIA Collective Communications Library            │
│   • AllReduce, AllGather, Broadcast                         │
│   • 拓扑感知                                                │
│   • 自动选择最优路径                                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
