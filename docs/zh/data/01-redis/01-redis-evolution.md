# 06-01: 一图看懂 Redis 演进史

## 核心观点
Redis 从简单缓存演进为多模态数据结构服务器。

## 演进图

```
┌─────────────────────────────────────────────────────────────┐
│                    Redis 演进                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   2009: Redis 诞生                                          │
│   ┌─────────────┐  单值存储, Redis = Remote Dictionary      │
│   │   Redis     │  Server                                   │
│   └─────────────┘                                          │
│                                                             │
│   2013: 数据结构扩展                                        │
│   ┌─────────────┐  Hash, List, Set, Sorted Set              │
│   │  数据结构   │  发布/订阅, 事务                          │
│   └─────────────┘                                          │
│                                                             │
│   2015: 集群化                                              │
│   ┌─────────────┐  Redis Cluster, 自动分片                  │
│   │  Cluster    │  主从复制, 故障转移                       │
│   └─────────────┘                                          │
│                                                             │
│   2020+: 模块化                                              │
│   ┌─────────────┐  Redis Stack                              │
│   │   Redis     │  • RedisJSON (文档)                       │
│   │   Modules   │  • RediSearch (搜索)                      │
│   └─────────────┘  • RedisGraph (图)                        │
│                    • RedisAI (AI 推理)                      │
│                                                             │
│   Redis 7: Streams, Functions, ACL                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
