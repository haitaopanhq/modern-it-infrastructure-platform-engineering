# 06-03: 一图看懂数据库分布式演进

## 核心观点
数据库分布式演进是从 ACID 到 BASE 的权衡史。

## 演进路径

```
┌─────────────────────────────────────────────────────────────┐
│                  数据库分布式演进                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   单一数据库 → 主从复制 → 分片 → NewSQL                     │
│                                                             │
│   ┌───────────┐    ┌───────────┐    ┌───────────┐          │
│   │  MySQL    │    │  TiDB     │    │  Cockroach│          │
│   │  Oracle   │───►│  Yugabyte │    │  PlanetScale│         │
│   │  PG       │    │  (NewSQL) │    │  (Serverless)│        │
│   └───────────┘    └───────────┘    └───────────┘          │
│                                                             │
│   NewSQL 特点:                                              │
│   • 强一致性 (ACID)                                         │
│   • 水平扩展                                                │
│   • 分布式事务                                              │
│   • 兼容 MySQL/PG 协议                                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
