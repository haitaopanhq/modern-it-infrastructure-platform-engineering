# 10-05: 一图看懂 OpenClaw 与 Agent Gateway 的意义

## 核心观点
OpenClaw 是 Agent 与基础设施的桥梁。

## OpenClaw 架构

```
┌─────────────────────────────────────────────────────────────┐
│                    OpenClaw 架构                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                   入口层                             │  │
│   │   Slack │ Discord │ Telegram │ WebChat │ 定时任务   │  │
│   └─────────────────────────────┬───────────────────────┘  │
│                                 │                           │
│   ┌─────────────────────────────┼───────────────────────┐  │
│   │                   Agent Gateway                      │  │
│   │   ┌──────────┐ ┌──────────┐ ┌──────────┐           │  │
│   │   │  会话    │ │  技能    │ │  工具    │           │  │
│   │   │  管理    │ │  匹配    │ │  执行    │           │  │
│   │   └──────────┘ └──────────┘ └──────────┘           │  │
│   └─────────────────────────────┬───────────────────────┘  │
│                                 │                           │
│   ┌─────────────────────────────┴───────────────────────┐  │
│   │                   工具层                             │  │
│   │   K8s │ GitHub │ Terraform │ 数据库 │ 消息          │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   核心能力:                                                  │
│   • 多渠道接入                                              │
│   • 技能/工具系统                                           │
│   • 记忆/会话管理                                           │
│   • 安全/审计                                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 与传统平台的对比

| 维度 | 传统平台 | Agent 控制面 |
|------|---------|-------------|
| 交互 | 图形/CLI | 自然语言 |
| 执行 | 手动触发 | 自动/对话 |
| 能力 | 单一工具 | 多模型协作 |
| 控制 | 规则引擎 | AI 决策 |
