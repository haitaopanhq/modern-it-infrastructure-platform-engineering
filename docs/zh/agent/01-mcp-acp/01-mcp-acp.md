# 10-01: 一图看懂 MCP/ACP

## 核心观点
MCP/ACP 是 AI 时代的 API 协议。

## 协议对比

```
┌─────────────────────────────────────────────────────────────┐
│                 MCP / ACP 协议                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   MCP (Model Context Protocol):                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │  Anthropic 提出                                      │  │
│   │  • 工具调用标准化                                    │  │
│   │  • 资源访问                                          │  │
│   │  • 提示模板                                          │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   ACP (Agent Communication Protocol):                       │
│   ┌─────────────────────────────────────────────────────┐  │
│   │  OpenAI 提出                                         │  │
│   │  • Agent 间通信                                      │  │
│   │  • 状态同步                                          │  │
│   │  • 任务协作                                          │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   对比传统 API:                                             │
│   ┌─────────────┬─────────────┬─────────────┐              │
│   │    维度     │  REST API   │  MCP/ACP    │              │
│   ├─────────────┼─────────────┼─────────────┤              │
│   │   面向     │   机器      │    Agent    │              │
│   │   调用方式 │   同步调用  │   异步协作   │              │
│   │   协议     │   HTTP/JSON │   JSON-RPC  │              │
│   │   能力     │   CRUD      │  Tool/Res   │              │
│   └─────────────┴─────────────┴─────────────┘              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
